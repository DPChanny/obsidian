## 1. 역할

현재 브랜치의 HEAD 위치를 특정 커밋으로 이동시키고, 인덱스와 작업 디렉토리를 재설정함.

## 2. 주요 모드

### 2.1. --soft

- **HEAD**: 이동함.
- **Index/Worktree**: 유지됨. (되돌린 커밋 내용이 스테이징된 상태임)

### 2.2. --mixed (기본값)

- **HEAD/Index**: 재설정됨.
- **Worktree**: 유지됨. (되돌린 커밋 내용이 스테이징되지 않은 상태임)

### 2.3. --hard

- **HEAD/Index/Worktree**: 모두 지정된 커밋 상태로 강제 재설정됨.
- **주의**: 저장되지 않은 모든 변경 사항이 영구 삭제됨.

## 3. 문법 요약

```bash
# 직전 커밋 취소 및 스테이징 유지
git reset --soft HEAD~1

# 특정 지점으로 강제 초기화
git reset --hard <commit-id>
```
